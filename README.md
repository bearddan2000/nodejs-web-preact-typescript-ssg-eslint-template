# nodejs-web-preact-typescript-ssg-eslint-template

## Description
A template for the preact framework a superset for react.

Feature
- ssg prerender
- eslint

## Tech stack
- npm
- preact
- ssg
- eslint

## Docker stack
- node

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
